# words_search:

`words_search` an algorithm for finding specified words in a two-dimensional
array

_Example:_

```
const WORDS = ['cat', 'cry', 'run', 'try', 'tree'];
```

```
const array =
   [
        ['i', 'y', 'l', 'e'],
        ['f', 'r', 'r', 'h'],
        ['q', 't', 'a', 'c'],
        ['r', 'u', 'n', 'z']
    ]
```

_It's possible to find following words: 'cat', 'cry', 'run', 'try' (but word
'tree' is not present)_
